⌥ -> ALT , ⌃ -> CONTROL , ⌘ -> COMMAND , ⇧ –> SHIFT

⌥ + ⌃ + ⌘ + 8 --> Mac invert color

Sublime

  • ⌘ / --> line comment
  • ⌘ ⌥ / --> block comment
  • ⌘ f --> search all dir
  • ⌘ ] --> indent
  • ⌘ K --> fold all
  • ⌘ J --> unfold all
  • Layout (Split Screen) → Alt + Cmd+1,2 …(1 pane, 2 pane etc.)
  • switch between tabs → cmd+1,2 …
  • view → Show sidebar → shows the folder hierarchy
  • view → Hide minimap (Removes the right pane showing the map of doc )
  • cmd+shift +f → global search

Chrome on MAC

  • Reopen Last 10 closed tabs → ⌘-Shift-T
  • Minimizes the window. → ⌘-M
  • Hides Google Chrome. → ⌘-H
  • Hides all other windows. → ⌘-⌥-H
  • Dev tools → ⌘-⌥
